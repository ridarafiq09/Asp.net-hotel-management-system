# ASP.NET Hotel Management System

A web-based **Hotel Management System** developed using **ASP.NET Web Forms**.  
The application provides core hotel management functionalities including user authentication, staff management, and reservation handling.

---

## Technologies Used

- ASP.NET Web Forms
- C#
- HTML5, CSS3
- Bootstrap
- SQL Server (Local Database)
- Visual Studio

---

## Application Features

- User login and registration
- Dashboard for system overview
- User profile management
- Hotel catalog management
- Staff management
- Reservation management
- Master Page layout for consistent UI
- Responsive design using Bootstrap
- Database-driven operations using SQL Server

---

## Project Structure

- `WebForms/` – Application pages
- `Content/` – CSS and styling files
- `Images/` – Application images
- `App_Data/` – Database configuration (excluded from Git tracking)
- `Properties/` – Project properties and configuration files
